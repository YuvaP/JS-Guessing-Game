Yuva Guessing Game with JavaScript.

Goal: Guess any number to find if you have selected the lucky number.

How does the game work?

  • The javaScript code generates the random number between 1 to 10 inclusive.
  • The user is prompted to guess that number.
  • The user's number is compared with the code generated number and verified.
  • The user is given 3 tries to guess that lucky number.
  • The input box will be disabled after the match or after 3 unsuccessful attempts.
  • Depending upon the outcome, the user gets the message.
